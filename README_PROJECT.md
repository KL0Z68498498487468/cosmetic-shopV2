# 🌟 Luxe Essence - Premium Parfumerie Shop

Премиум e-commerce сайт парфюмерии класса люкс с потрясающими анимациями, видео и современным дизайном.

## 🚀 Быстрый старт

### Установка зависимостей
```bash
npm install
```

### Запуск разработки
```bash
npm run dev
```
Откройте [http://localhost:5173](http://localhost:5173)

### Продакшн-сборка
```bash
npm run build
npm run preview
```

## 📁 Структура проекта

```
src/
├── components/
│   ├── layout/
│   │   ├── Header.jsx          # Шапка с навигацией и корзиной
│   │   ├── Footer.jsx          # Подвал
│   │   └── Layout.jsx          # Главный layout
│   └── ui/
│       ├── ProductCard.jsx     # Карточка товара (3D тильт)
│       ├── BottlePreview.jsx   # 3D превью флакона
│       ├── FilterSidebar.jsx   # Фильтры каталога
│       └── ThemeToggle.jsx     # Переключатель темы
├── pages/
│   ├── Home.jsx                # Главная со видео
│   ├── Catalog.jsx             # Каталог с фильтрами
│   ├── Product.jsx             # Страница товара
│   ├── Brands.jsx              # Бренды (со видео)
│   ├── Profile.jsx             # Личный кабинет
│   ├── Cart.jsx                # Корзина
│   ├── About.jsx               # О бренде (со статистикой)
│   ├── Limited.jsx             # Лимитированные (с таймером)
│   └── NotFound.jsx            # 404 страница
├── hooks/
│   ├── useTheme.js             # Управление темой
│   ├── use3DTilt.js            # 3D эффект мыши
│   ├── useScrollAnimation.js    # GSAP scroll анимации
│   └── useCountUp.js           # Счётная анимация чисел
├── store/
│   ├── store.js                # Zustand хранилища
│   └── data.json               # Mock данные
├── App.jsx                     # Роутинг
└── index.css                   # Tailwind + кастомные стили
```

## ✨ Ключевые фишки

### 1. **Тёмная/Светлая тема**
- Плавное переключение с морфингом иконок
- Сохранение в localStorage
- Кастомный скроллбар

### 2. **3D Эффекты**
- Вращение флакона за мышкой на главной
- 3D тильт на карточках товаров
- Parallax эффект при скролле

### 3. **Анимации**
- Framer Motion для переходов компонентов
- GSAP для сложных scroll-анимаций
- Confetti эффект при добавлении в корзину
- Счётная анимация для статистики

### 4. **Видео**
- Фоновое видео на главной странице
- Видео на странице Brands
- Responsive видео с плейсхолдерами

### 5. **Фильтрация**
- По нотам аромата (категориям)
- По цене
- По бренду
- Активные фильтры с возможностью удаления

### 6. **Управление состоянием**
- Zustand для корзины, избранного и темы
- Синхронизация со всеми компонентами

### 7. **Адаптивный дизайн**
- Mobile-first подход
- Медиа-запросы для всех breakpoints
- Мобильное меню в Header

## 🎨 Цветовая палитра

## 🔧 Технологический стек

- **React 19.1.1** - Функциональные компоненты
- **Vite 7.3.2** - Build tool
- **Tailwind CSS 4** - Стилизация
- **Framer Motion 12.20** - Основные анимации
- **GSAP 3.12.5** - Продвинутые анимации
- **Zustand 4.5** - Управление состоянием
- **React Router 6.28** - Роутинг
- **Lucide Icons** - Иконки
- **Canvas Confetti** - Конфетти эффект

## 📱 Адаптивность

### Breakpoints
- `sm`: 640px
- `md`: 768px
- `lg`: 1024px
- `xl`: 1280px

Все компоненты полностью адаптивны и тестированы на всех размерах экранов.

## 📊 Страницы и функциональность

| Страница | Описание |
|----------|---------|
| `/` | Главная с видео, 3D флаконом, лучшими товарами |
| `/catalog` | Каталог с фильтрами, сортировкой, сеткой товаров |
| `/product/:id` | Детальная информация о товаре, пирамида нот |
| `/brands` | Сетка брендов-партнёров с видео фоном |
| `/profile` | Личный кабинет, история заказов, избранное |
| `/cart` | Корзина с редактированием количества |
| `/about` | История бренда, статистика с анимацией |
| `/limited` | Лимитированные коллекции с таймером |
| `*` | 404 с красивой анимацией |
